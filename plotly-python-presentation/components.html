What is Plotly?


A Python library

Plotly's Python graphing library makes interactive, publication-quality graphs.

But why would we need it?

Easier readability of statistics incoming from our projects üëç

Installation process

Plotly may be installed using:

  • PIP $ pip install plotly
  • Poetry $ poetry add plotly
  • Conda $ conda install -c plotly

Note: Pandas needs to be installed as Plotly requires it.

Install it: pip install pandas

Lets construct our first graph

import plotly.express as px 
fig = px.bar(x=["a", "b", "c"], y=[1, 3, 2]) 
fig.write_html('first_figure.html', auto_open=True)
